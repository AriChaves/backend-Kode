# 📅 Smart Calendar

An intelligent Android planner built with Kotlin that uses AI to optimize 
your daily schedule and task management.

## ✨ Features
- 🤖 AI-driven task organization based on free time slots
- 📊 Daily grade calculator and academic tracker
- ⏰ Smart reminders and schedule suggestions
- 📱 Clean and intuitive mobile UI

## 🛠️ Tech Stack
- Kotlin & Android Studio
- AI API integration
- Material Design UI
